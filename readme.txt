WP FollowMe CSS plugin allows you to add a twitter "Follow me" badge on your WordPress blog.

== Description ==

This plugin is a fork of the original WP FollowMe by wpBurn. Like the original WP FollowMe CSS allow you to add a twitter "Follow me" badge on your WordPress blog.

WP FollowMe CSS has been ported to CSS3 and so no longer require Flash, allowing it to work on phones and any other modern browser.

= With FollowMe CSS you can =

* Change the color of the badge
* Change the follow me label
* Change the position of the badge (left side or right side, and the distance from top)

= Demo =
You can see a demo here: [UbuntuDanmark.dk](https://ubuntudanmark.dk/)


== Installation ==

1. Extract all files from the zip archive to your wp-content/plugins folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Add your Twitter URL and configure other options as you wish
